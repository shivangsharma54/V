# Countdown To Her Special Day

A beautiful, aesthetic countdown website dedicated to Vaishnavi, counting down to January 1st, 2026.

## Features

- **Countdown Timer**: A real-time countdown showing days, hours, minutes, and seconds.
- **Photo Gallery**: A grid gallery displaying cherished memories.
- **Aesthetic Design**: Uses a dark, modern theme with glassmorphism effects and smooth animations.
- **Mouse Trail**: A magical glitter and heart emoji trail follows your cursor.
- **Responsive**: Fully responsive design that looks great on mobile and desktop.

## File Structure

- `index.html`: The main structure of the website.
- `style.css`: All the styling, animations, and responsive rules.
- `script.js`: Logic for the countdown timer and mouse trail effect.
- `images/`: Folder containing the photos for the gallery.

## How to Run

Simply open `index.html` in any modern web browser.

## Customization

- **Countdown Date**: Edit the date in `script.js`.
- **Images**: Add or replace images in the `images/` folder and update `index.html`.
- **Styling**: Modify `style.css` to change colors, fonts, or animations.
